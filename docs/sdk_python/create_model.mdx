## Description

The `get_model()` and `create_model()` methods enable you to get an existing model or create and deploy a new model.

## Syntax

Use the `get_model()` method to get an existing model:

```bash
my_model = project.get_model('my_model')
```

Or, the `create_model()` method to create and train a new model:

```bash
my_model = project.create_model (
    name = 'my_model',
    predict = 'target',
    query = my_table
)
```

Here is the syntax to create an OpenAI model:

```bash
sentiment_classifier = project.create_model(
      name='sentiment_classifier',
      engine='huggingface',
      predict='sentiment',
      options={
          'model_name':'cardiffnlp/twitter-roberta-base-sentiment',
          'input_column': 'comment',
          'labels': ['negative', 'neutral', 'positive']
      }
)
```
